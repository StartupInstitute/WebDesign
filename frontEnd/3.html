Let's Talk CSS

Cascading Style Sheets

  • Used for “presentation” of your HTML

  • Controls the styles of your page

  • Also Controls the layout of your page (where stuff is positioned)

  • Can have different styles for different media formats (eg. print, screen, mobile)

What is a style?

We Pick a Selector

(A selector is the part of our HTML we want to style, for example, h1, h2, p or li)

h1 { color:red; }
h2 { font-size:20px; }
a { text-decoration:none; }

You can also pick id or class names, as well as multiple tags- but for now let's keep it simple

We Make a Declaration

(using CSS properties and values we add formatting)

p { color:red; }
h2 { font-size:20px; }
a { text-decoration:none; }

Combining Selectors

You can add styles to multiple selectors

Like all the h1, h2, & h3s

Us a comma (,) for multiple selectors

.h1, .h2 { color:red; }

Or specify certain selectors

Like only links inside a nav

Can “Cascade” styles with spaces between selectors

HTML
<nav ><li> ... </li></nav>

CSS
nav li  { color:red; } /* only p's inside of outer */

Lots of Psudeo-classes too

a:hover { color:blue; } /* only links we are hovering over */

a:visited { color:red; } /* links we've visited already */

p:first-child { color:red; } /* the first paragraph in a container */

How do we use this in our sites?

Your CSS goes in 1 document

often named style.css

You link to this document in each HTML page's head




This goes in Your HTML

<head>

  <link href="stylesheet.css" rel="Stylesheet" type="text/css" />

</head>
